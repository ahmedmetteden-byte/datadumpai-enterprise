# DataDumpAI

AI-powered business reporting in minutes.

Upload your business documents, generate professional reports, and download them ready to share.

## Version

1.0

## What it does

1. **Upload documents** — PDF, Word, Excel, PowerPoint, CSV, and text files
2. **Generate reports** — Executive Summary, Board Report, Management Report, Financial Analysis, Meeting Minutes
3. **Ask Copilot** — Summarize documents, surface risks, and review recommendations
4. **Download** — Export reports as PDF, DOCX, or Markdown

## Run locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

Set `OPENAI_API_KEY` in a `.env` file in the project root.
